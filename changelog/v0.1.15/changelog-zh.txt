LinkEmby v0.1.15 更新日志

发布日期: 2025-11-04

🔧 重构优化

  • 仪表盘性能优化: 将多个 API 调用整合为单一统一接口，减少网络请求次数，提升页面加载速度
  • Emby API 调用优化: 使用 Items/Counts 端点替代单独的图书馆项目计数，显著减少 API 调用次数
  • 统计报告缓存优化: 在许可证管理器中添加 24 小时 TTL 缓存，提高数据获取效率

🎉 新功能

  • Emby API 查询缓存: 实现基于可配置 TTL 的缓存机制，减少重复 API 查询，提高资源分配效率
  • 代理连接测试功能: 添加代理配置测试功能，允许管理员在保存前验证代理连接状态和性能
