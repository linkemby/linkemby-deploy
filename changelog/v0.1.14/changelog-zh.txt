LinkEmby v0.1.14 更新日志

发布日期: 2025-11-01

🔧 重构优化

  • 服务器操作重构: 增强了类型安全性和错误处理能力,完善了所有操作中的类型定义,改进了管理员操作、用户操作和身份验证操作的错误处理和验证机制,统一了响应类型,提升了代码组织和可维护性

  • UI 组件优化: 增强了数据表格组件的操作处理,改进了响应式设计和移动端布局,优化了全局 CSS 间距工具,提升了用户仪表板和 Emby 服务显示效果

  • Markdown 渲染统一: 使用统一的 MarkdownRenderer 组件替换了所有不一致的 Markdown 渲染器,启用了 HTML 支持,减少了代码重复,确保了应用中的一致性渲染

🎉 新功能

  • 按需媒体请求系统: 实现了完整的媒体按需请求功能,用户可浏览最新电影和电视节目,支持通过 TMDB 集成发现热门内容,提供了搜索和媒体类型筛选,用户可选择电视剧季集,管理员可审核和处理请求,系统根据订阅限额自动处理请求

  • 订阅系统增强: 改进了许可证管理器的错误处理,为订阅计划增加了按需配额字段,实现了订阅计划模态框改进,增加了订阅功能和限额的验证,支持按计划的动态功能切换

  • 全局代理系统: 为外部 API 请求实现了代理系统,支持 HTTP 和 SOCKS5 代理协议,支持代理身份验证,集成了 Redis 缓存来存储代理配置,与 TMDB 等外部 API 调用集成,为外部服务访问提供了灵活的网络支持

  • 用户按需界面: 为用户实现了按需功能界面,提供了媒体浏览和搜索功能,支持请求历史查看和状态跟踪,包含了趋势媒体浏览、搜索功能和配额显示,用户可预览媒体详情和提交请求,支持取消待审核请求和查看管理员备注,全面支持订阅计划集成

🐛 问题修复

  • 演示数据初始化: 改进了演示数据初始化脚本,现在在清理过程中保留许可证代码,避免了数据重置时丢失许可信息,增强了用户体验
