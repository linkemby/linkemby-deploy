LinkEmby v0.2.3 更新日志

发布日期: 2025-11-23

🎉 新功能

  • Cloudflare Turnstile 人类验证: 为登录和注册页面添加 Turnstile 验证,提升账户安全性

  • 用户播放统计: 新增用户播放历史统计页面,支持个人观看记录查看和数据可视化

  • 点播系统改进: 优化内容请求状态显示,避免重复请求,提升用户体验

🔧 重构优化

  • 许可证管理优化: 重构许可证管理器,改进许可证页面 UI 显示,优化订阅状态管理

  • 配置和环保支持: 更新环境配置示例,改进 Docker 环境设置,增强用户管理和密码修改功能

  • 统计监控增强: 完善异常检测机制,支持可配置的历史数据过滤,统一缓存命名规范

  • API 错误处理: 改进 Emby API 错误处理机制,增强 WASM 加载器容错能力

🐛 问题修复

  • 修复订阅自动续费: 解决自动续费设置不生效的问题,完善订阅弹窗验证逻辑

  • 修复服务器备份 URL 支持: 改进服务器备份 URL 相关功能
